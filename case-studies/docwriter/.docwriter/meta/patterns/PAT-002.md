## Insight

First-attempt tasks applied more invariants on average (5.7 invariants) vs two-cycle update tasks. TASK-008 applied 11 invariants and succeeded first try. TASK-010 applied 13 invariants and needed only standard 2 cycles.

## Evidence

- Run 2 (DOC-3137): First-attempt tasks averaged 13.1 invariants vs multi-cycle tasks 6.7. The 3 highest invariant counts (T-008: 24, T-007: 18, T-012: 29) all achieved first-attempt acceptance. Strongly confirmed.
- Acceptance rate across 2 runs: confirmed
- Promoted: low → medium (2nd pipeline run confirmation)

Encourage writers to explicitly apply more invariants upfront rather than relying on reviewer feedback.
